# Guide d'Utilisation - OPAYS HQ

## 1. Introduction
OPAYS HQ est l'outil central de gestion pour les associés d'OPAYS TECH. Il permet de coordonner les ventes, le développement technique et la gouvernance.

## 2. Guide par Rôle

### 👤 Associé Ventes (Business Dev)
- **Action :** Chaque nouveau contact doit être entré dans le module "Leads".
- **Suivi :** Mettre à jour le statut du lead après chaque échange.
- **Objectif :** Atteindre le statut "CLOSED_WON" pour déclencher la mise en production.

### 💻 Associé Technique (CTO)
- **Action :** Une fois le lead gagné, créer un projet technique lié.
- **Suivi :** Définir la stack technique et les échéances de livraison.
- **Collaboration :** Utiliser Antigravity pour accélérer le développement des automatisations.

### 🛡️ COO (Gestion & Admin)
- **Action :** Vérifier les factures et les paiements.
- **Suivi :** Valider les "Milestones" pour libérer les parts sociales correspondantes dans le tracker.

## 3. Système d'Invitations
Pour ajouter un associé :
1. Allez dans le menu **Paramètres > Équipe**.
2. Cliquez sur **Générer un lien d'invitation**.
3. Choisissez le rôle (CTO, Sales, etc.).
4. Envoyez le lien généré. Le lien expire après 7 jours.

## 4. Espace Audit IA (Le Cœur de l'App)
C'est ici que la magie opère.
- **Upload :** Chargez les données brutes du client.
- **Analyse :** L'IA traite les données selon le skill `efficiency_engineering`.
- **Export :** Le rapport est généré automatiquement et peut être envoyé directement au client depuis l'app.

## 4. Maintenance de l'Outil
L'application est hébergée sur Cloudflare et utilise Supabase pour la base de données. En cas de bug, contacter l'IA de bord (Antigravity).
