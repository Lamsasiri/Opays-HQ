# OPAYS HQ

## Introduction
OPAYS HQ est l'application de gestion interne d'OPAYS TECH, un cabinet d'ingénierie de l'efficience basé en RDC. 

Cette application permet de coordonner les ventes, le développement technique, la gouvernance des associés et dispose d'un espace dédié aux audits d'entreprise assistés par IA.

## Tech Stack
- **Frontend :** Next.js 15 (App Router)
- **Styling :** Tailwind CSS + Framer Motion (Studio Design)
- **Backend :** Supabase (Auth, Database, Storage)
- **IA :** Intégration directe avec l'agent Antigravity pour les audits.

## Structure
- `/src/components` : Composants UI premium (Dashboard, Audit Space).
- `/supabase` : Schémas SQL et configuration.

## Installation
```bash
npm install
npm run dev
```
