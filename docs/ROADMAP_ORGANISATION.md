# 🗺️ Plan d'Action & Structuration de l'Équipe Opays Tech

Pour qu'un projet de cette envergure avance, chaque membre doit avoir un **"périmètre de victoire"** clair. Voici une structuration optimisée pour votre équipe de 5, utilisant les outils que nous venons de bâtir.

---

## 👥 1. Répartition des Rôles (The "Who does What")

| Profil | Rôle Stratégique | Outil Principal sur Opays HQ |
| :--- | :--- | :--- |
| **Vous (Lead Tech)** | Architecture & Vision | **Workspace** (Gouvernance technique) |
| **Evans (Dev)** | Développement & Ops | **Workspace** (Builds & Déploiements) |
| **Commerciale 1** | Prospection & Pipeline | **Sales Hub** (Leads & CRM) |
| **Commerciale 2** | Customer Success & ROI | **Sales Hub** (Calculateur ROI & Preuves) |
| **Juriste** | Compliance & Contrats | **Settings** (Documents Associés) & **Contracts** |

---

## 🛠️ 2. Tâches Spécifiques & Concrètes (Next 30 Days)

### 💻 Pôle Technique (Vous + Evans)
*   **Evans** : Connecter le Workspace à vos dépôts GitHub réels pour voir les logs de commit en direct dans la console.
*   **Evans** : Finaliser le module "Preview" pour que les commerciaux puissent montrer des démos aux clients.
*   **Vous** : Définir la "Definition of Done" pour les projets IA et configurer les templates de tâches dans l'onglet Projets.

### 💰 Pôle Commercial (Les 2 Filles)
*   **Fille 1 (Prospection)** : Identifier 10 entreprises cibles en RDC et les ajouter dans l'onglet **Leads**. Remplir les colonnes "Valeur Estimée".
*   **Fille 2 (Analyse)** : Pour chaque Lead, lancer une simulation dans le **Calculateur ROI** du Studio. Enregistrer les résultats pour que vous puissiez les montrer en rendez-vous.
*   **Objectif Commun** : Transformer 2 Leads en "WON" (Projets actifs) d'ici la fin du mois.

### ⚖️ Pôle Légal & Administratif (Le Juriste)
*   **Rédaction** : Finaliser le Pacte d'Associés officiel d'Opays Tech.
*   **HQ Ops** : Utiliser le module **"Documents Personnalisés"** (dans Settings) pour uploader le pacte signé et les pièces d'identité de chacun des 5 membres.
*   **Contrats** : Créer le template de contrat de prestation standard dans l'onglet **Contrats** pour les futurs clients.

---

## 📈 3. Comment utiliser Opays HQ pour piloter l'équipe ?

1.  **Le Lundi (Briefing)** : Regardez ensemble le **Dashboard Global**. Qui a de nouvelles tâches ? Quel est le solde de la trésorerie ?
2.  **Le Mercredi (Sales Check)** : Les commerciales présentent les simulations ROI du Studio. Le Juriste vérifie si les contrats sont prêts pour ces prospects.
3.  **Le Vendredi (Build)** : Evans et vous validez les avancées techniques dans le Workspace et passez les tâches terminées en "DONE".
